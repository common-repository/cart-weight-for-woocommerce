== Description ==

Display the product and cart weight in cart page, checkout page, order email and admin side.

= WHY WE BUILT THIS PLUGIN? =

Default WooCommerce does not provide the weight-related information for your order.

Once a customer adds products to the cart then, they can get notified that how much your order weight is.

Based on the cart weight, they can get an idea of how the order delivery option they can select like local pickup or need shipping, etc.

Users can easily get the idea of how much cart contains the weight and also notify in the email as well once place an order.

Plugin Demo: <a href="http://pluginsdemo.thedotstore.com/cart-weight-for-woocommerce/" target="_blank"> View Demo </a>

= Key Features =

* Display weight per product on the cart page
* Display total cart weight on the cart page
* Display total cart weight on the checkout page
* Display total cart notice after adding product to the cart.
* Display cart weight on admin order detail.
* Display cart weight on the user order email.
* Display cart weight information on My Account in order detail page for users.
* Display cart weight on Mini cart.
